# 🌦️ Weather App

A simple and beginner-friendly weather app built using HTML, CSS, and JavaScript.  
This project allows users to search for any city and view real-time weather information using the Open-Meteo API.

I built this project to practice working with APIs, JavaScript async functions, DOM manipulation, and creating responsive web applications.

---

##  Features

- Search weather by city name
- Real-time weather updates
- Weather condition icons
- Temperature display
- Wind speed information
- Local date and time
- Error handling for invalid city names
- Press Enter key to search
- Clean and responsive design

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Open-Meteo API
